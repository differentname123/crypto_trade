# 任务 (Task)
**目标**：为指定的 Biance 广场陌生人帖子生成两条用于推广优质带单组合的高转化率互动评论。
**受众**：浏览该帖子的完全陌生的用户。
**验收标准**：
1. 评论必须最大概率促使陌生浏览者产生好奇或利益诉求，从而完成跟单链接的点击。
2. 针对同一输入，输出两套异构方案：“带单员本人”视角与“已跟单用户”视角。
3. 文本必须精准拆分为“评论正文”与“超链接文本”，拼接后语意连贯、无割裂感。
4. 提供基于真实转化概率的 0-10 分量化评估。

# 背景 (Context)
待推广的带单策略经过精挑细选，真实业绩数据优异，具备极高的分享与跟单价值。
我们的引流场景为 Biance 广场的陌生帖子评论区。原帖作者既不是带单员也不是跟单用户。
前端最终的展现形式为：`[评论正文]` + `[超链接文本]`。调用系统会自动将 `[超链接文本]` 渲染为高亮且可跳转跟单页面的链接。

# 约束 (Constraints)
1. **唯一优化标尺（最高优先级）**：所有推演与生成的唯一目标是“最大概率提升跟单链接点击率与实际转化率”。修辞、礼貌、逻辑铺垫等若与该目标冲突，必须无条件让位。
2. **动态策略搜索（禁红线）**：禁止使用任何生硬的营销套路、刻板人设（如“专业说教”或“刻意感恩”）。必须根据 `<data>` 中的帖子内容、语境与评论区氛围独立推演，自主选择最易引发转化的切入点（探讨、顺从、科普、自嘲、悬念等）。
3. **双视角绝对异构**：“带单员本人”与“已跟单用户”的转化策略必须有本质差异，以覆盖不同受众心理。
4. **极简约束**：总文本长度必须控制在移动端信息流“一瞥即读完”的范围内。不追求句式平滑，砍掉所有不能促成点击的无效字符，追求单位字数内极高的推荐说服力。
5. **超链接文本硬性规定**：
   - **字数**：严格控制在 2~6 个字符以内。
   - **结构**：必须置于尾部，带有轻微动作暗示或指向性。
   - **衔接**：绝对不能有割裂感。超链接文本必须作为评论正文语意的自然延伸，两者组合必须是一句完整、通顺的话。
   - **红线**：禁止使用“点击链接”、“跟单主页”等生硬机械词汇，必须使用高爆点字词。
6. **客观转化潜力评分（0-10分）**：必须给出量化评分。评估标准仅为“毫无背景知识的陌生人看到后点击的概率”。若原帖为排斥外部信息的“营销沙漠”（如极度悲愤维权、纯技术排外帖），必须直接给出低分，禁止强行粉饰高分。
7. **输出红线**：你的全部输出必须是严格的 JSON 格式字符串，不能包含任何 JSON 之外的文本，也不能使用 Markdown 代码块（如 ```json ）包裹，以便系统直接解析。
